# 🍋 Little Lemon - Booking Page

This is a simple React app for booking a table at Little Lemon restaurant. Built with:

- React
- Tailwind CSS
- Vite (build tool)

## 🚀 Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/little-lemon-booking.git
   cd little-lemon-booking
   ```
